### Dependencies

- PHP 7.0+

### Run

1 -) Clone
```bash
$ git clone https://github.com/thiiagoms/flood
```

2 -) Give permissions
```bash
$ cd flood
$ chmod +x flood
```

3-) Run:
```bash
$ ./flood
```
